### selected, value和key

`selected`表示当前选择项，`value`表示当前选择值。`key`表示数据项的键，默认为`'id'`。

它们三者的关系如下：`selected[key] == value`。其中`selected`和`value`是联动的，当一项变化时会同时改变另一项。

*注意：暂时不支持`value`赋初始值。

<div class="m-example"></div>

```xml
<jr-form>
    <jr-col span=6>
        <jr-tree-view  source={source} selected={selected} />
    </jr-col>
    <jr-col span=6>
        <jr-tree-view  source={source} value={value} />
    </jr-col>
</jr-form>
<p>选择的分别是是: {selected.name} {value}</p>
```

```javascript
var component = new JRUI.Component({
    template: template,
    data: {
        source: [
            {id: 1, name: '母婴儿童'},
            {id: 2, name: '美容彩妆'},
            {id: 3, name: '服饰鞋包'}
        ]
    },
    config: function() {
        this.data.selected = this.data.source[0];
    }
});
```

### 禁用某一项，禁用组件

<div class="m-example"></div>

```xml
<jr-row>
    <jr-col span=6>
        <jr-tree-view source={source} value={value} />
    </jr-col>
    <jr-col span=6>
        <jr-tree-view source={source} disabled />
    </jr-col>
</jr-row>
<p>选择的是：{value}</p>
```

```javascript
var component = new JRUI.Component({
    template: template,
```

### 分隔线

<div class="m-example"></div>

```xml
<jr-tree-view source={source} value={value} />
<p>选择的是：{value}</p>
```

```javascript
var component = new JRUI.Component({
    template: template,
    data: {
        source: [
            {name: '家居个护', checked: true, children: [

                {name: '居家用品', disabled: true, children: [
                    {name: '杯子'},
                    {name: '锅具刀具'}
```
